# Manual Canvas AI

Manual Canvas AI 是一个面向产品说明书制作场景的 AI 在线画板，目标是替代传统 Word/WPS 在说明书排版、多语言翻译和版本维护中的低效流程。

项目支持 A4、A5 以及自定义页面尺寸，用户可以在浏览器中像使用设计工具一样编辑说明书页面，包括文本、图片、表格、警示信息、步骤说明和产品参数等内容。同时，系统会结合 AI 模型对说明书内容进行结构化解析、术语抽取、多语言翻译、版面适配和质量检查，帮助用户更高效地生成多语言产品说明书。

## Tech Stack

- Frontend: React 18
- Backend: Python FastAPI
- Database: PostgreSQL
- Message Queue: RabbitMQ
- AI: Mimo / LLM-powered agents
